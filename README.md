# Go Logger

A lightweight, zero-dependency logging library for Go that provides colored output for better readability.

## Features

* **Simple:** No configuration required.
* **Colored Output:** Visual distinction between Info, Warning, and Error logs.
* **Timestamps:** Automatically adds `[YYYY/MM/DD HH:MM:SS]` to every log.

## Installation

Run the following command in your project folder:

```bash
go get github.com/KubeX3/go-logger
```
